GitHubトークンの取得方法

1トークン作成画面を開く

GitHubアプリまたはブラウザで
Settings → Developer settings → Fine-grained tokens → Generate new token
を開きます。

2対象リポジトリを選ぶ

Repository access で「All repositories」または更新したいリポジトリを選択します。有効期限(Expiration)も設定します。

3権限(Permissions)を設定

Repository permissions で次の3つを設定してください。

・Contents: Read and write(必須)
・Metadata: Read only(自動で付与)
・Workflows: Read and write(.github/workflows を更新する場合)

4トークンをコピーして貼り付け

「Generate token」を押すと github_pat_… で始まるトークンが1回だけ表示されます。コピーして、このアプリの設定画面に貼り付けて保存してください。

※トークンは画面を閉じると再表示できません。失くした場合は再生成してください。
